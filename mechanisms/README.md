# Mechanism Packs

This folder contains built-in mechanism packs consumed by CADAI retrieval/UI.

## Included packs

- `core-open-v1` (CC0-1.0, built-in core patterns)
- `curated-cq-gears-apache` (Apache-2.0, inspired by [cq_gears](https://github.com/meadiode/cq_gears))
- `curated-cq-warehouse-apache` (Apache-2.0, inspired by [cq_warehouse](https://github.com/gumyr/cq_warehouse))
- `curated-cadquery-examples-apache` (Apache-2.0, inspired by [CadQuery examples](https://github.com/CadQuery/cadquery/tree/master/examples))

## Notes

- Entries are prompt/reference patterns, not bundled third-party source code.
- External pack import is supported via `install_mechanism_pack(manifest_url)` and enforced SPDX allow-list.
